# 个人博客网站

这是一个使用 Next.js 构建的个人博客网站，用于记录学习笔记和个人心得。

## 技术栈

- Next.js 14
- React 18
- TailwindCSS
- TypeScript

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 部署

本项目可以直接部署到 GitHub Pages。使用以下命令构建项目：

```bash
npm run build
```

## 目录结构

```
├── components/     # React 组件
├── pages/         # 页面文件
├── public/        # 静态资源
├── styles/        # 样式文件
└── posts/         # 博客文章 (Markdown)
```
